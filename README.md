# ChessLiteJs
This is a chess-like, turn-based game implemented in JavaScript. It demonstrates basic programming concepts, game logic, and handling of user input.
## Files
- `project.html` – main HTML file to open the game in a browser  
- `project.js` – contains the game logic and piece movement
